DemonLegal+ Mod

DemonLegal+ is a fully Pointercrate Demon List legal mod for Geometry Dash 2.2 designed to provide helpful, fair tools without cheating.

Features include:

  • Input display overlay
  • Practice mode helpers
  • FPS and frame timing info
  • Customizable UI
